# Toolon
Multi-tool desktop application built using Java Swing with multiple utilities and tools
## Features

Time Tools
- Analog Clock
- Digital Clock
- Stopwatch

Utilities
- Calculator
- Currency Converter
- Temperature Converter

Productivity
- Notepad
- To-Do List
- Contact Book

Games
- Tic Tac Toe
- Guess Game
- Quiz App

## Tech Stack
- Java (Swing)
- Maven

## How to Run

mvn clean install
java -jar target/Dashboard-1.0-SNAPSHOT.jar
